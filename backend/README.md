init.sql Ele serve como a "planta baixa" ou o projeto arquitetônico do seu banco de dados. Em vez de você ir lá no pgAdmin e criar tabela por tabela clicando com o mouse, você escreve tudo em código SQL nesse arquivo.
Histórico e Segurança: O código fica salvo no Git. Se você errar algo no banco de dados e precisar apagar tudo, você não perde a estrutura.
